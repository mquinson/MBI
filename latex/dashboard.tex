      \textbf{Metric}&\textbf{Definition and meaning}\\\hline

      Specificity&\multirow{1.3}{*}{Ability to not find errors in correct codes.  $S=\frac{TN}{TN + FP}$}\\[4pt]\hline
      Recall&\multirow{1.3}{*}{Ability to find existing errors. $R=\frac{TP}{TP+FN}$}\\[4pt]\hline
      Precision& \multirow{1.3}{*}{Potential confidence when a code is reported as correct. $P=\frac{TP}{TP+FP}$}\\[4pt]\hline
      F1 Score&\multirow{1.3}{*}{Overall bug-finding quality. $F1=\frac{2 \times P \times R}{P+R}$}\\[4pt]\hline
      Overall accuracy&\multirow{1.3}{*}{Proportion of correct diagnostics over all tests. $A=\frac{TP+TN}{CE+RE+TO+TP+FP+TN+FN}$}\\[4pt]\hline
